# React App Boilerplate

## Using Libraries
- react
- redux
- react-dom
- react-redux
- redux-thunk
- react-route
- axios
- immutablejs

## Prerequisite
- hosts 설정
/etc/hosts 상에 아래 내용 추가
```
127.0.0.1 local.reactapp.com
```

## To start
```
npm install
npm start
```
브라우저에서 local.reactapp.com:4000 접속
